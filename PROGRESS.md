## 🛠 최근 해결된 버그 / 개선 사항
- [x] **화면 레이아웃 찌그러짐 해결**: Tailwind CSS v4 테마 변수 충돌 (`--spacing-md`가 `max-w-md`에 영향 주는 문제) 수정.
- [x] **헤더 모바일 너비 한정**: 헤더가 전체 화면이 아닌 모바일 레이아웃(`max-w-md`) 안에서 중앙 정렬되도록 조정.
- [x] **리뷰 탭 UI 레이아웃 통일**: 리뷰 작성 페이지(/review/create)에 전역 Header 및 BottomNavigation을 노출하고 배경색과 버튼 레이아웃을 홈 화면에 맞추어 일관되게 조정.
- [x] **필터 UI 전면 개편**: 홈 화면에서 독립된 '가격', '거리순' 버튼을 제거하고 카테고리 드롭다운 내에 정렬/가격 필터를 통합. 드롭다운 잘림(overflow) 버그 수정.
- [x] **가격 필터 기준 단순화**: 3단계($/$$/$$$ 기호) 기준을 '만원 이하'(under_10k) / '만원 초과'(over_10k) 2단계로 전환하여 직관성 향상.
- [x] **홈 화면 불필요 버튼 제거**: 리스트 헤딩 우측 '거리순 정렬' 버튼 및 우하단 AI 플로팅(FAB) 버튼 삭제.
- [x] **취소 버튼 시인성 개선**: 상황 변경 모달의 '취소' 버튼 색상을 비활성 필터 칩과 동일한 스타일로 통일.
